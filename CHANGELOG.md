# Change Log

All notable changes to the "telegram-emoji-preview" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.11]
- Improved regex to support whitespace around `=` in attributes (e.g. `icon_custom_emoji_id = VAR`).

## [1.0.10]
- Added support for curly braces `{VAR}` in variable syntax (e.g. for React/JSX).

## [1.0.9]

- Added support for variable resolution in `icon_custom_emoji_id`.

## [1.0.8]

- Re-published with correct version.

## [1.0.7]

- Added support for `icon_custom_emoji_id` format.

## [Unreleased]

- Initial release
